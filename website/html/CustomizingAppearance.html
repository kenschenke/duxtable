Customizing Appearance

The appearance of <DuxTable> is controlled entirely using CSS. Every component uses a CSS class that can be overridden. For a complete list of the CSS rules see the src/duxtable.css file in the GitHub.

If you are using webpack, it might be tempting to simply import the duxtable.css file. This is perfectly fine but will not allow you to override the CSS. This is because the CSS is applied by React during component creation. See the Getting Started topic for more information on using the <link> element to reference the CSS in a way that it can be overridden.

The default CSS styles are designed for a "light mode" web site, but <DuxTable> can be styled using CSS. The example shows a table in the default "light mode". You can toggle on "dark mode" to see how a table can be integrated into different designs, such as this site.
