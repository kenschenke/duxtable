Fixed-Header Scrolling Tables

<DuxTable> tables can scroll the contents of the table body with the column headers remaining in place. The table could potentially display a large number of rows. Please keep browser performance in mind. There are a few properties to set for a fixed-header table.

Property Default Value Description
pagination true Set this property to false to disable pagination
bodyHeight (none) This is how many pixels high to make the visible portion of the table body.
