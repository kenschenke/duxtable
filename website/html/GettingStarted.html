Getting Started

If you're the ready, fire, aim type (aren't all programmers?), you should find everything you need on this page and the topics under DuxTable Basics to start using DuxTable.

Installation

    npm install duxtable

You also need to link to the duxtable.css file in the node_modules/duxtable/website/dist/ directory. 

    <link rel="stylesheet" href="duxtable.css"/>
You might be tempted to use the duxtable.css file in your project with webpack and an import statement. This will work as long as you don't need to override any of the CSS in the file.

Redux Store

DuxTable stores table state in the Redux store. It requires the root duxtable object in the store. A reducer is supplied to manage this. 

    import {DuxTableReducer} from 'duxtable';

    const appReducer = combineReducers({
        ...  // other reducers
        duxtable: DuxTableReducer
    });

Browser Requirements

DuxTable requires a browser with support for CSS FlexBox. It has been tested on the modern evergreen browsers (Chrome and Firefox) as well as Internet Explorer 11. It definitely won't work on Internet Explorer 10 or earlier.
