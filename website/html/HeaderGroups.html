Header Groups

<DuxTable> supports header groups. Multiple headers can be visually grouped to help emphasize column groupings. Groups are specified using the groupHeaders property as an array of objects. Each object has the following properties.

Property Description
title The title of the group header
startCol The starting column (column numbers start at 0)
endCol (optional) The ending column (column numbers start at 0). If not specified, this group ends with the last column of the table.
