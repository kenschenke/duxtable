Helper Functions

A few helper functions are available which simplify extracting values from the Redux store for a table.

getSelectedRow

    const row = getSelectedRow(state, tableName);

This function is passed the current Redux application state and the name of the table and returns the Id from the currently selected row or undefined if no row is currently selected. See the topic on Single and Multi Selections for an example of use.

getSelectedRows

    const rows = getSelectedRows(state, tableName);

This function is passed the current Redux application state and the name of the table and returns an array of the Ids from the currently selected rows. See the topic on Single and Multi Selections for an example of use.

getTableStoreValue

    const value = getTableStoreValue(state, tableName, propertyName, defaultValue);

This function is passed the current Redux application state, the name of the table, the name of the property, and the default value if the property is not defined.
