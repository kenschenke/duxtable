Hidden Columns

Columns in the table can be hidden or shown based on the dynamic needs of the application or the width of the browser window. This is handled with the hidden property in the columns array supplied to DuxTable.

The hidden property for a column can be a boolean value, a function that returns a boolean value, or an object that specifies whether or not a column is hidden based on width of the browser window. The width breakpoints are the same as defined in Bootstrap 4. The breakpoints are listed below. Following is a sample columns array. 

    const isColumnHidden = () => {
        return true;
    }

    const columns = [
        {
            title: 'Column 1',
            field: 'col1value',
            hidden: false
        },
        {
            title: 'Hidden Column',
            field: 'hiddenvalue',
            hidden: isColumnHidden
        }
    ];

Responsive Breakpoints

A column can be dymanically shown or hidden based on the width of the browser window. This is done by specifying an object with boolean values for various widths. The breakpoints are the same as those in Bootstrap 4. These breakpoints are:

Breakpoint Description
xs Extra small devices (portrait phones, less than 576px)
sm Small devices (landscape phones, 576px and up)
md Medium devices (tablets, 768px and up)
lg Large devices (desktops, 992px and up)
xl Extra large devices (large desktops, 1200px and up)

A breakpoint object might look something like this: 

    const hidden = {
        xs: true,
        sm: true,
        md: false
    };

Notice the hidden object is missing lg and xl. If a size breakpoint is missing from the object, the breakpoint for the next smallest screen size is used. In this case, browsers with large or extra-large windows would fall back to the medium breakpoint. The column would be hidden on small and extra-small browser windows. If the browser window is the xs size and that breakpoint is missing from the object, the next lowest breakpoint is used instead.

In this example, the columns hide at browser widths more narrow than their breakpoint. For example, the Column Xl will be hidden when the browser window is smaller than xl. Try resizing the browser window to see the columns hide and show.
