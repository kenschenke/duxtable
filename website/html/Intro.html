Welcome To DuxTable!

A React / Redux table component with resizable columns, searching, and sorting.

DuxTable is a React / Redux table with built-in support for sorting, searching, resizable columns, and fixed column headers.

Why Another React Table?

DuxTable offers the following notable features.

  • Real-time searching of table cells
  • Sortable columns by clicking on the column headers
  • User can resize columns
  • Fixed column headers with scrolling
  • Build-in pagination
  • Fully customizable with CSS
  • Single row and multiple row selections
  • Custom cell rendering
  • Table footers
  • Header groups
