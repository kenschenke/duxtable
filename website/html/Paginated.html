Paginated Tables

<DuxTable> comes with built-in support for pagination. The entire set of rows is provided in the data property and DuxTable splits it into manageable pages for the user. There are a few properties which impact pagination behavior.

Property Default Value Description
pagination true If true, rows are paginated
pageSize 10 This is how many rows are shown per page
If the last page does not contain enough rows to fill the page, empty rows are shown so the table has a uniform height. This keeps the pagination controls from jumping up and down as the user moves between pages.
