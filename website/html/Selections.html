Single and Multi Selections

<DuxTable> supports single and multi-row selections through the selectionMode property. The possible values are "none" (the default), "single", or "multi".

The selected row or rows are available in the Redux state for the table in the selectedRows property. This is an array of the values from the selected row or rows. The value property is specified with the rowKey property to DuxTable. For example, if the table had a list of United States, the data array could look like this: 

    const states = [
        {label: 'Alaska', value: 'AK'},
        {label: 'Arkansas', value: 'AR'},
        ...
    ];

    const cols = [
        {field: 'label', title: 'State'},
        ...
    ];
The DuxTable would look like this: 
    <DuxTable columns={cols} rowKey="value" .../>
